Alphabetize Icons
================

Alphabetizes icons for the canvas-lms project

# Usage

Install:
```bash
git clone https://github.com/fivetanley/alphabetize_icons.git
cd alphabetize_icons
npm install
node index.js
```

Running:

Copy the html you want to sort into 'data.html' after cloning the repo.

`node index`
